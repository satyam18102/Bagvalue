<!-- # 🛒 Shoppy – A Complete React Native E-commerce App (Expo)

Shoppy is a stylish, dark-mode-supported e-commerce mobile app built using **React Native**, **Expo**, and **AsyncStorage**. It features:
- 🔐 Authentication (Signup, Login with validations)
- 🏠 Home with banners, categories, search & recommended items
- 📦 Product listing & detail view with wishlist, cart, quantity selector
- 🛍 Cart management, checkout, order tracking
- 🧑‍💼 Profile view/edit, dark mode toggle, past orders, wishlist
- 💙 Fully optimized for both Mobile & Web via Expo

## 🔧 Tech Stack

- **React Native + Expo**
- **React Navigation** (Stack, Tab)
- **Context API + useReducer** (Cart & Wishlist)
- **AsyncStorage** – Local storage for persistence
- **Formik + Yup** – Form handling and validation
- **Axios** – API calls (`https://fakestoreapi.com`)
- **Expo ImagePicker** – Profile image support

## 🌟 Features

✅ Signup & Login with validations  
✅ Home with search, banners, and auto-scroll offers

✅ Checkout with address & payment method  
✅ Order tracking, cancel, delivered  
✅ Wishlist management  
✅ Profile editing and dark mode toggle  
✅ Alerts & toast notifications  
✅ Works on Android, iOS & Web
